# Hammer Starter

一个优雅的项目快速启动工具，帮助你快速创建各种类型的项目。

## 功能特点

- 🚀 快速创建项目
- 🎨 美观的终端界面
- 📦 支持多种项目模板
- ⚡️ 简单易用的命令行工具

## 安装

```bash
npm install -g hammer-kits
```

## 使用方法

```bash
npx hammer-kits init
```

## 支持的项目模板

- WxT-Vue 项目
- Starter-Uno 项目

## 开发

```bash
# 安装依赖
npm install

# 链接到全局
npm link
```
